# Joomla GSoC Test Plugin - Testtitle

## Description
This plugin demonstrates the use of Joomla 5 content plugins with com_ajax.

## Features
- Adds a configurable text field parameter in plugin settings
- Loads in Joomla article form view (administrator)
- Uses com_ajax to retrieve parameter value
- Automatically sets article title field

## Technologies Used
- Joomla 5 Plugin System
- com_ajax
- JavaScript (Fetch API)

## Installation
1. Download ZIP
2. Install via Joomla Extension Manager
3. Enable plugin
4. Set "Default Title Text"

## Usage
Go to:
Content → Articles → New

The title field will be auto-filled using plugin parameter.
